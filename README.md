       T Test trên 2 biến số: Kiểm định giả thuyết về giá trị trung bình (đối với mẫu nhỏ) H0: trung bình mẫu 1 = trung bình mẫu 2
       Proportion Test: Từ mẫu nhỏ suy ra mẫu lớn
